# Assignment Instruction

This instruction is used to updade properties and attributes of an object with new values.

When a single destination is modified, the source must be an expression with a single result that conforms, or can be converted to, the destination.

This instruction can also be used to assign multiple destinations at once. In that case, the source expression must have at least as many results as there are destinations, and assignment is performed in order: the n<sup>th</sup> result is assigned to the n<sup>th</sup> listed destination. The source must of course conform, or convert, to the destination.
